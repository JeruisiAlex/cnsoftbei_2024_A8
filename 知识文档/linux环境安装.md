### 初始化

​	更新安装器版本指令：sudo apt update



### 编译器

#### gcc

​	安装指令：sudo apt install gcc

​	版本查看指令：gcc --version

#### g++（用于编译运行 c++）

​	安装指令：sudo apt install g++

​	版本查看指令：g++ --version

### 软件安装

#### git

​	安装指令：sudo apt-get install git

​	版本查看指令：git --version

#### freerdp

​	安装freerdp包的指令：sudo apt install freerdp2-dev
